  2. 深度學習的輸入與輸出

人類大腦約有1000億神經元。每個神經元有多個樹突，一個軸突。上一個神經元的軸突與下一個神經元的樹突，通過神經遞質傳遞信息。

模擬神經元的樹突與軸突： z = a1w1 + ... + akwk + ... + aKwK + b

模擬神經遞質的作用： a = σ(z)

這種對神經遞質的模擬叫激活函數（非線性）。激活函數有很多，比如： sigmoid(z)， tanh(z)， softplus(z)， relu(z)

模擬神經元的樹突與軸突的計算： z = 2 x 1 + (-1) x (-2) + 1 x (-1) + 1 = 4

模擬神經遞質的作用的計算： a = σ(4) = sigmoid(4) ≈ 0.98

模擬將神經元組合爲神經網絡：

layer1 = σ(input x W1 + b1)
layer2 = σ(layer1 x W2 + b2)
...
output = σ(layerL x WL + bL)

深度學習模擬神經網絡的運算（激活函數使用relu）

深度學習的輸入之圖像，其在計算機內都是以數字矩陣的形式存儲的。 輸入是將圖像的像素值（0~255或0~1）按順序輸入神經網絡，輸出則爲圖像的每一個類標（label）的概率。 通常我們選擇概率最大的類標，作爲圖像的最終預測值。

深度學習主要使用的就是神經網絡，這裏的“深度”指的就是神經網絡的層數（通常大於等於三層）。 神經網絡主要是模擬人腦神經系統，其中輸入輸出是需要數字化的，這樣計算機纔可以處理。
