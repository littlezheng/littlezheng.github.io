第一次組裝電腦記錄

  • 2017-02-18
    • 記錄

    最近在微信公衆號上看到一篇文章，講的是怎麼使用大約 1000 美刀，組裝一臺入門級的深度學習機器。 看完這篇文章後心動了，也想嘗試一下，所以就有了第一次組裝電腦記錄。

    首先感謝京東的 裝機大師，節省了好些挑選硬件的時間。

    話不多說，下面列出我選擇的硬件配置。 這些硬件都是買自京東，價格也來自於京東，人民幣（¥）與美元（$）的匯率參考，$1 = ¥6.8640。

  • CPU，處理器

    • 英特尔（Intel）酷睿四核I5-7500 盒装CPU处理器

    ¥1579.00

    Intel 的 i7 和 i5 的價格相差大概一千多元。其型號結尾帶 k 表示可以超頻。 從性能上來說，i5 對我來說足夠了，我也不需要超頻。所以我選擇的是較新的 i5-7500。 安裝的時候注意，有兩個防呆卡扣，不要裝反了。 還有就是盒裝 CPU 自帶的風扇可用於不超頻的 CPU，如果使用超頻的 CPU，最好買一個獨立的 CPU 散熱器。

  • GPU，顯卡

    • 华硕（ASUS）DUAL-GTX1060-O6G 1569-1809MHz 6G/8GHz GDDR5 PCI-E3.0显卡

    ¥2099.00

    NVIDIA 常見的第十代顯卡有 GTX1080(8G)，GTX1070(8G)，GTX1060(6G/3G)，GTX1050(2G) 等，從高到低，價格相差在一千五到兩千元之間。 所謂的一分價錢一分貨，其實想買 GTX1070 的，不過因爲預算關係，最終還是買了 GTX1060(6G)。 拆裝顯卡的時候注意，主板上的 PCI-E 卡槽上有一個卡扣，不要硬掰。

  • 內存

    • 金士顿(Kingston)骇客神条 Fury系列 DDR4 2400 16G 台式机内存

    ¥939.00

    考慮到內存的擴展性，買了一條 16G 的內存，而不是兩條 8G 的。 注意查看主板是否支持 DDR4。

  • 硬盤

    • 希捷(SEAGATE)酷鱼系列 2TB 7200转64M SATA3 台式机硬盘(ST2000DM006)

    ¥449.00

    硬盤現在都白菜價了，直接選擇 2T，7200 轉的硬盤。

  • 主板

    • 华硕（ASUS）PRIME B250-PRO 主板（Intel B250/LGA 1151）

    ¥949.00

    主板型號有 Z 系列和 B 系列的區別，Z 系列支持超頻。 板型大小有 ATX，MTX(M-ATX)，ITX(mini-ITX)等，尺寸依次減小，需要根據主板板型大小選擇機箱大小。 同時需要注意，主板的 CPU 接口型號要和 CPU 針腳型號相適應，這裏我選擇的是 LGA 1151。 主板是一個整體，需要考慮對各個芯片是否支持，然後考慮其擴展性，接口是否豐富等。

  • 電源

    • 海韵（Seasonic）额定650W G-650 电源（80PLUS金牌/半模组/全日系电容/支持Haswell/支持背线）

    ¥699.00

    電源相當於電腦的心臟，所以需要認真對待，並留有功率餘量，以期將來擴展之用。 再者，需要查看電源的接口是否支持主板，CPU，顯卡等。

  • 機箱

    • 酷冷至尊(CoolerMaster)毁灭战将U3版 黑色 中塔式机箱(支持ATX主板/USB3.0/SGCC板材)

    ¥209.00

    機箱的選擇餘量很大，各種類型都有，只要型號與主板向適應即可。

  • 顯示器

    • LG 22M35AN-B? 21.5英寸爱眼滤蓝光LED背光液晶显示器

    ¥649.00

    顯示器看個人選擇，預算關係，選擇了這款 LG 的顯示器。

  • 裝機工具

    • 惠泽(Huzo) 五金工具螺丝刀套装多功能手机维修起子组拆机工具 (45件套/HZ-4501)

    ¥59.90

    主要使用十字螺絲刀，這套工具以後也有用處。

    最後是操作系統的安裝。Windows 10 就不說了，這裏講講安裝 Ubuntu 16.14 系統。 不曉得爲什麼，不能使用 Ubuntu 桌面版，一開始選擇安裝，顯示器就沒信號。 所以這裏，我選擇 Ubuntu 服務器版。使用服務器版需要安裝的時候需要插網線下載一個桌面。 不過也好，可以不用選擇 Ubuntu 自帶的桌面，我這裏選擇的是 GNOME 桌面，看着也很舒服。

    因爲第一次組裝電腦，光硬件安裝就花了四個多小時，不過裝過一次就熟悉好多了。 主要是系統，第一次嘗試一臺電腦只裝 Ubuntu 系統，希望使用愉快吧，Have fun!
